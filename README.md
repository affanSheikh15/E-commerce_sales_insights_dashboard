# E-Commerce Sales Insights Dashboard

An end-to-end **Data Analytics and Business Intelligence project** that analyzes e-commerce sales data to uncover key business insights and support data-driven decisions.

---

## Project Overview

This project focuses on analyzing global e-commerce sales data to help businesses understand:

* Which regions or countries generate the highest revenue
* Which products perform best
* How sales trends change over time
* The impact of discounts or campaigns

**Goal:** To empower business teams with clear, data-driven insights that improve sales strategies, marketing focus, and operational decisions.

---

## Technologies & Tools Used

* **SQL** – Data extraction, cleaning, and aggregation
* **Python** – Data preprocessing, exploratory analysis (EDA), and visualization
* **Power BI** – Interactive dashboard creation for key metrics and trends
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn (if used)



## What This Project Demonstrates

* Discovering hidden sales patterns and seasonal trends
* Creating insightful dashboards for business stakeholders
* Building a complete data pipeline: Raw Data → Clean Data → Insights → Dashboard
* Translating business questions into data-backed answers


## How to Explore / Run

1. Run the SQL queries to extract and clean the raw sales data.
2. Use Python scripts to perform EDA and generate visualizations.
3. Open the Power BI file to explore the interactive dashboard with filters and KPIs.
4. (Optional) Replace the dataset path in scripts if using your own data.
